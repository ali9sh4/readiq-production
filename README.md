# Readiq

Next.js (App Router) production codebase for Readiq.

## Mobile API

This codebase exposes a parallel `/api/*` surface for the upcoming React Native mobile app, in addition to the existing web pages and server actions. See [`docs/MOBILE_API_MIGRATION.md`](docs/MOBILE_API_MIGRATION.md) for the full plan and current status, and [`docs/MOBILE_API_TESTING.md`](docs/MOBILE_API_TESTING.md) for endpoint testing recipes. A live status board lives at [`docs/MOBILE_PROJECT_STATE.md`](docs/MOBILE_PROJECT_STATE.md). The mobile app lives in a separate repo (TBD).
